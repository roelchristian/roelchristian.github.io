Definition and examples

Let $X$ be a set (whose elements we shall call ‘points’). A metric on $X$ is a function $\rho: X \times X \to \mathbb{R}$ such that for all $x, y, z \in X$,

  1. $\rho(x, y) \geq 0$ (non-negativity)
  2. $\rho(x, y) = 0$ if and only if $x = y$ (identity of indiscernibles)
  3. $\rho(x, y) = \rho(y, x)$ (symmetry)
  4. $\rho(x, z) \leq \rho(x, y) + \rho(y, z)$ (triangle inequality)

The pair $(X, \rho)$ is called a metric space. With some abuse of notation, we may refer to $X$ as a metric space with the implied metric $\rho$.

Examples.

Metrics in $\mathbb{R}$. Consider the set $\mathbb{R}$ of real numbers. The absolute difference function $|\cdot - \cdot|: \mathbb{R} \times \mathbb{R} \to \mathbb{R}$ is a metric on $\mathbb{R}$. Indeed for any real numbers $x, y, z$, we have $|x - y| \geq 0$, $|x - y| = 0$ if and only if $x = y$, $|x - y| = |y - x|$, and $|x - z| \leq |x - y| + |y - z|$.

Let $n \geq 1$. The set $\mathbb{R}^n$ is the set of all $n$-tuples of real numbers. For the sake of brevity, we shall denote an element of $\mathbb{R}^n$ by $\mathbf{x} = (\xi_1, \xi_2, \ldots, \xi_n)$, where $\xi_i \in \mathbb{R}$ for $1 \leq i \leq n$. We define the Euclidean metric on $\mathbb{R}^n$ to be the map ${\ell^2}: \mathbb{R}^n \times \mathbb{R}^n \to \mathbb{R}$ given by $$ {\ell^2}(\mathbf{x}, \mathbf{y}) = \sqrt{\sum_{i = 1}^n (\xi_i - \eta_i)^2}. $$

In $\mathbb{R}^3$ and $\mathbb{R}^2$ one can visualize ${\ell^2}(\mathbf{x}, \mathbf{y})$ as the distance between the points $\mathbf{x}$ and $\mathbf{y}$ in the Euclidean plane and space, respectively. The Euclidean metric is the most commonly used metric on $\mathbb{R}^n$ as it is the natural generalization of the absolute difference function on $\mathbb{R}$ and provides a natural notion of distance between points in $\mathbb{R}^n$.

The Manhattan metric (also known as the taxicab metric) on $\mathbb{R}^n$ is the map ${\ell^1}: \mathbb{R}^n \times \mathbb{R}^n \to \mathbb{R}$ given by $$ {\ell^1}(\mathbf{x}, \mathbf{y}) = \sum_{i = 1}^n |\xi_i - \eta_i| $$ for $\mathbf{x} = (\xi_1, \xi_2, \ldots, \xi_n)$ and $\mathbf{y} = (\eta_1, \eta_2, \ldots, \eta_n)$. The Manhattan metric is so-called because it is the distance a taxicab would have to travel in a rectangular grid of streets (similar to those found in Manhattan) to get from the point $\mathbf{x}$ to the point $\mathbf{y}$.

Finally, we define the Chebyshev distance on $\mathbb{R}^n$ as $$ {\ell^\infty}(\mathbf{x}, \mathbf{y}) = \max_{1 \leq i \leq n} |\xi_i - \eta_i|. $$

These three metrics are related by the inequalities $$ {\ell^\infty}(\mathbf{x}, \mathbf{y}) \leq {\ell^2}(\mathbf{x}, \mathbf{y}) \leq {\ell^1}(\mathbf{x}, \mathbf{y}) \leq n {\ell^\infty}(\mathbf{x}, \mathbf{y}). $$

Discrete metric. Let $X$ be an arbitrary set. The discrete metric on $X$ is the map $\rho: X \times X \to \mathbb{R}$ given by $$ \rho(x, y) = \begin{cases} 0 & \text{if } x = y \\ 1 & \text{if } x \neq y. \end{cases} $$
