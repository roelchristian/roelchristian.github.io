A note on the design of this site

One of the reasons why I decided to abandon Wordpress was because of how clunky it could be. My original page was never interaction-heavy, as it’s main purpose was (and still is) to be a place where I can display the stuff I wrote or made. Thus the decision to make the whole site static. Thus Jekyll. I have tried to make this site as simple and as minimalistic as I could, and to some success I’d like to think.

The main css file for this site has been forked from the Tufte-CSS project, which is itself inspired by the design philosophy of the American statistician Edward Tufte. I found it while searching for CSS frameworks that did a decent job dealing with footnotes (I can be fond of footnotes) and fell in love with how elegantly it made footnotes (read: sidenotes) appear for the web.
